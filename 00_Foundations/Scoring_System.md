# Scoring System

## Dimensions

| Metric | Weight |
|--------|-------:|
| Strategic Clarity | 20 |
| Differentiation | 15 |
| User Value | 20 |
| Creative Quality | 15 |
| Feasibility | 10 |
| Consistency | 10 |
| Scalability | 10 |

Overall Score = Weighted Average

### Confidence Levels

95–100 : High confidence
80–94  : Good confidence
60–79  : Medium confidence
<60    : Insufficient information
