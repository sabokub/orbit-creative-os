# Architecture

## System Layers

1. User Interface
2. Orchestrator
3. Specialist Agents
4. Orbit Brain
5. Knowledge Library
6. Output Engine

## Data Flow

User Request
→ Intent Detection
→ Context Retrieval
→ Agent Selection
→ Parallel Reasoning
→ Critical Review
→ Final Synthesis

## Principles

- Orbit Brain is the only persistent source of truth.
- Agents never store state locally.
- Every output references previous decisions when applicable.
