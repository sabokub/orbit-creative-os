# Example: Coffee Shop

## Project Type

Hospitality brand.

## Goal

Create a complete concept for a coffee shop with a distinct visual and spatial identity.

## Audience

People who choose places for atmosphere, taste, social value and visual memory.

## Strategic Direction

The concept should avoid generic warm minimal coffee shop codes.

## Creative Direction

- Strong point of view
- Specific material palette
- Clear signature moments
- Simple but memorable service logic
- Visual consistency across space, menu and social channels

## Test Tasks

1. Define positioning.
2. Create a spatial narrative.
3. Create a visual territory.
4. Draft a launch content plan.
