# Example: Interior Studio

## Project Type

Interior design and creative direction studio.

## Goal

Create a brand and visual system for a studio that designs spaces with strong lifestyle and editorial value.

## Audience

Urban clients who care about taste, image, culture and the emotional quality of their home.

## Strategic Direction

The brand should not feel like a generic decoration service. It should feel like a spatial direction studio.

## Creative Direction

- Editorial lifestyle
- Lived-in interiors
- Strong material presence
- Controlled colour accents
- Human details
- Photo-ready corners

## Test Tasks

1. Create a brand positioning.
2. Define a visual territory.
3. Create a living room image brief.
4. Review the output with Orbit Critic.
