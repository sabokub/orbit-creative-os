# Content Workflow

## Purpose

Create a repeatable content system from brand strategy.

## Steps

1. Read Orbit Brain
2. Define audience stage
3. Select content pillar
4. Select format
5. Draft content brief
6. Adapt to channel
7. Review consistency
8. Plan reuse
9. Track performance

## Inputs

- Brand positioning
- Audience
- Voice
- Offer
- Goal

## Outputs

- Pillars
- Series ideas
- Calendar
- Briefs
- Reuse plan
