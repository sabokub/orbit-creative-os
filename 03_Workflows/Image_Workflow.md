# Image Workflow

## Purpose

Turn creative direction into clear visual briefs.

## Steps

1. Read Orbit Brain
2. Identify visual goal
3. Select visual territory
4. Define subject and context
5. Define composition
6. Define light and materials
7. Build image brief
8. Create variations
9. Review results

## Inputs

- Visual Territory
- Visual DNA
- Material Language
- Prompt Architecture

## Outputs

- Master brief
- Variation briefs
- Constraints
- Review checklist
