# 24March Command Center — Alpha

ORBIT remains the reasoning and automation engine. The 24March Command Center is the visual operating layer used to run the studio at a glance.

## Included in this alpha

- Responsive desktop sidebar and mobile bottom navigation
- Live launch countdown to 1 September 2026
- Project, output, review and export metrics
- Dynamic project progress based on the ORBIT pipeline stage
- Launch priority board
- ORBIT engine launcher
- Visual Brand Brain page
- Guided project intake
- Redesigned project workspace
- Redesigned AI workflow runner with explicit human validation
- Redesigned output library
- Shared rounded, graphic and editorial UI system

## Product rule

The interface should feel like a creative cockpit, not a generic SaaS dashboard and not a Notion clone. Information must be visible quickly, while every generated output remains reviewable and manually validated.

## Next product layer

The current launch priorities are interface-level defaults. A future data model can make the launch board, daily priorities and studio-level tasks editable and persistent in Redis, alongside ORBIT projects.
