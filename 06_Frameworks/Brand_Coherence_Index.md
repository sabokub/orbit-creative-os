# Brand Coherence Index

## Purpose

Measure whether a brand presents the same meaning across every touchpoint.

## Touchpoints

- Website
- Social media
- Email
- Packaging
- Photography
- Language
- Environment

## Scoring

0 = absent
1 = weak
2 = acceptable
3 = strong
4 = excellent

## Output

- Overall score
- Strongest touchpoints
- Weakest touchpoints
- Priority fixes

## Rule

A brand with weak coherence should fix the system before producing more assets.
