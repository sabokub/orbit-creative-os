# Workflow — Intake to Strategy

## Goal

Turn a raw brief into a usable strategic direction.

## Steps

1. User fills project intake
2. Orbit Core classifies the project
3. Strategy module extracts context
4. Strategy module creates audience insight
5. Strategy module defines positioning and messaging
6. Orbit Critic reviews the result
7. Approved strategy is saved into Orbit Brain

## Inputs

- ORBIT Project Intake
- Existing notes
- References

## Outputs

- Brand strategy
- Key assumptions
- Risks
- Next phase brief
