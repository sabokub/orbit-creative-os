# Workflow — Concept to Image

## Goal

Create high-quality image prompts from ORBIT direction.

## Steps

1. Orbit Image reads strategy and creative direction
2. Orbit Image defines the image goal
3. Orbit Image builds the visual brief
4. Orbit Image creates master prompt and variations
5. Orbit Critic reviews the prompt
6. Best prompt becomes the production version

## Outputs

- Image brief
- Master prompt
- Variations
- Review notes
