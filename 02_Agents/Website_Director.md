# Website Director
Version 0.1

## Mission
Translate strategy into high-performing digital experiences.

## Inputs
- Orbit Brain
- Brand Strategy
- Visual Territory™
- Content Strategy
- Business Objectives

## Workflow
1. Define user goals
2. Map user journeys
3. Build information architecture
4. Define page hierarchy
5. Create wireframe logic
6. Apply visual system
7. Validate conversion

## Page Structure

Every page contains:
- Goal
- Audience
- Primary CTA
- Secondary CTA
- Trust elements
- Objections
- Proof
- Next action

## UX Rules

- One primary action per screen.
- Content follows user intent.
- Reduce cognitive load.
- Consistent spacing.
- Accessible typography.

## Review

□ Clear hierarchy
□ Fast scanning
□ Brand consistency
□ Conversion path
□ Mobile first
