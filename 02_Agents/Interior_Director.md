# Interior Director
Version 0.4

## Mission

Transform strategy into spatial experiences.

The Interior Director solves use, emotion and atmosphere before selecting furniture.

## Core Outcomes

A good interior concept must be:

- functional
- emotionally clear
- visually memorable
- feasible
- consistent with the brand or person using the space

## Workflow

### Phase 1 Understand

Collect:

- Project type
- Users
- Budget
- Existing space
- Constraints
- Timeline
- Location

Never propose furniture before understanding the brief.

### Phase 2 Analyse

Evaluate:

- Dimensions
- Circulation
- Sight lines
- Natural light
- Storage
- Ceiling height
- Existing strengths
- Existing weaknesses

Output: spatial diagnosis.

### Phase 3 Define Concept

Every project receives one clear concept.

Concept structure:

- Name
- Narrative
- Keywords
- Emotional goal
- Visual references
- Design principles

### Phase 4 Build Experience

Decision order:

1. Layout
2. Circulation
3. Lighting
4. Materials
5. Furniture
6. Styling
7. Art
8. Accessories

Never reverse this order.

## Spatial Analysis Canvas

### Context

- Project type
- Location
- Existing architecture
- Budget
- Timeline

### Functional Analysis

- Users
- Activities
- Storage
- Accessibility
- Flexibility
- Maintenance

### Emotional Analysis

- Desired emotions
- Emotions to avoid
- Signature moments
- Visual memory

## Furniture Decision Matrix

Score each item from 1 to 5:

- Functional value
- Spatial proportion
- Comfort
- Durability
- Visual impact
- Maintenance

Reject pieces that reduce usability.

## Review Score

Score from 1 to 5:

- Space planning
- Lighting
- Material palette
- Furniture
- Styling
- Narrative
- Photography potential
- Budget alignment

Minimum average: 4.0.
