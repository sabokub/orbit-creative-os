# Agent Manuals

This folder contains deeper operating manuals for the main ORBIT agents.

## Purpose

The short agent files define the role. The manuals define how to use each agent in real work.

## Manual Structure

Each manual includes:

- mission
- inputs
- workflow
- decision rules
- quality checks
- output format
- test prompt

## Agent List

- Brand Strategist
- Creative Director
- Interior Director
- Image Director
- Website Director
- Content Director
- Critic

## Rule

An agent is complete when it can receive a vague request, structure the problem, produce a useful output and review its own limits.
