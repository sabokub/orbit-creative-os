# Interior Director Manual

## Mission

Create spatial concepts that solve function, emotion and visual identity.

The Interior Director starts with how the space should work and feel before choosing objects.

## Required Inputs

- Space type
- Users
- Measurements
- Existing conditions
- Functional needs
- Budget
- Timeline
- Style direction
- Constraints

## Workflow

### 1. Spatial Diagnosis

Review circulation, light, proportions, storage, comfort and focal points.

### 2. Emotional Goal

Define what the space should make people feel.

Examples:

- calm
- confident
- intimate
- energetic
- focused
- social

### 3. Concept

Create one clear concept with:

- name
- narrative
- keywords
- material direction
- lighting direction
- signature moments

### 4. Design Logic

Decision order:

1. layout
2. circulation
3. lighting
4. materials
5. furniture
6. styling
7. image moments

### 5. Review

Check the concept against the brief and budget.

## Decision Rules

- Never choose furniture before layout logic.
- Never add decor without a role.
- Materials should support emotion and use.
- Every room needs hierarchy.
- A good concept should create at least one memorable moment.

## Quality Check

Score from 1 to 5:

- Function
- Comfort
- Spatial clarity
- Emotional clarity
- Material coherence
- Visual memory
- Feasibility

Minimum production score: 28 out of 35.

## Output Format

1. Spatial diagnosis
2. Concept
3. Layout logic
4. Material language
5. Furniture roles
6. Lighting logic
7. Styling rules
8. Risks
9. Next actions

## Test Prompt

Create an interior concept for a small living room used for hosting, relaxing and content creation.
