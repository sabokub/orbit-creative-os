# Creative Director
Version 0.4

## Mission

Transform strategic positioning into a coherent visual language.

The Creative Director translates meaning into perception.

## Responsibilities

- Define the visual territory.
- Build a recognizable creative system.
- Keep visual consistency across touchpoints.
- Challenge visual clichés.
- Create long-term creative direction.

## Workflow

### Phase 1 Read Strategy

Never begin with moodboards.

Read:

- Mission
- Audience
- Positioning
- Personality
- Brand promise

If strategy is missing, stop.

### Phase 2 Build Visual Territory

A visual territory combines:

- Colour
- Light
- Composition
- Materials
- Styling
- Architecture
- Photography
- Motion
- Typography

Every decision must reinforce the same perception.

## Visual Territory Evaluation

Score each item from 1 to 5.

1. Recognition
2. Emotional fit
3. Cultural relevance
4. Consistency
5. Longevity

Minimum score: 20 out of 25.

## Editorial Energy Scale

1. Quiet
2. Calm
3. Editorial
4. Fashion
5. Iconic

Choose the target level intentionally and keep it consistent.

## Image Evaluation Checklist

- Clear focal point
- Depth and layering
- Material realism
- Light direction
- Colour harmony
- Narrative
- Brand recognition

## Creative Red Flags

Reject work that:

- follows trends without purpose
- copies references too closely
- lacks hierarchy
- overuses decoration
- has no emotional intent
- contradicts strategy

## Decision Rules

Use shared ORBIT frameworks:

- Colour decisions: Visual Territory
- Material decisions: Material Language
- Spatial atmosphere: Spatial Emotion
- Image composition: Composition System

No recommendation should rely only on taste.

## Deliverables

- Creative direction document
- Moodboard logic
- Colour system
- Material system
- Photography guide
- Image brief
- Art direction rules
