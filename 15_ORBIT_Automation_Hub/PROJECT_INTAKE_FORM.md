# Project Intake Form

Formulaire de brief prêt à copier dans ChatGPT/Claude, ou à recréer champ pour champ dans Tally/Typeform (voir `SETUP.md` Option A).

Étend `13_ChatGPT_Runtime/Orbit_Project_Intake.md` avec les champs nécessaires au flow site + contenu + images + review.

## Formulaire

```text
## Projet

Nom :
Type d'activité :
Stade actuel (idée / lancement / relance) :

## Activité

Description en 2-3 phrases :
Offres / services proposés :

## Audience

Qui est le client cible (âge, profil, métier, style de vie) :
Ce qui les fait hésiter aujourd'hui :
Ce qu'ils doivent ressentir en découvrant la marque :

## Positionnement souhaité

En une phrase, ce que la marque est vraiment (pas juste ce qu'elle vend) :
Ce qui la différencie des concurrents :

## Direction visuelle

3 à 5 mots-clés de vibe :
Références (comptes, marques, moodboards, liens) :
À éviter absolument :
Couleurs d'accent à conserver (si existantes) :

## Canaux

Site internet : oui / non
Instagram : oui / non
TikTok : oui / non
Pinterest : oui / non
Autres :

## Contraintes

Budget de production :
Deadline :
Outils déjà utilisés (Notion, Canva, Figma, CMS...) :
Format de livraison attendu (Markdown / Google Doc / PDF) :

## Livrables demandés (cocher un ou plusieurs)

[ ] Stratégie de marque
[ ] Direction créative
[ ] Structure + copywriting du site
[ ] Prompts image du site
[ ] Calendrier de contenu réseaux
[ ] Captions, hooks, idées reels
[ ] Prompts image des réseaux
[ ] Review critique

## Critère de succès

Comment on saura que c'est réussi :
```

## Utilisation

1. Copier ce bloc.
2. Le remplir pour le projet.
3. Le coller en premier message dans une conversation ORBIT (après `13_ChatGPT_Runtime/Start_Here.md`), ou l'utiliser comme payload d'entrée pour un des master prompts dans `prompts/`.

Voir `examples/24March_Studio_Project.md` pour un exemple rempli.
