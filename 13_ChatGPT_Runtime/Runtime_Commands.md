# Runtime Commands

Use these commands inside ChatGPT to run ORBIT faster.

## Core Commands

### /orbit-start
Start a new ORBIT project from scratch.

### /orbit-route
Analyse the request and choose the right module.

### /orbit-brief
Create or complete the project brief.

### /orbit-review
Send the current output through Orbit Critic.

### /orbit-next
Choose the next best action.

## Module Commands

### /brand
Use Orbit Brand.

### /creative
Use Orbit Creative.

### /interior
Use Orbit Interior.

### /image
Use Orbit Image.

### /website
Use Orbit Website.

### /content
Use Orbit Content.

### /critic
Use Orbit Critic.

## Example

```text
/orbit-route
I need a visual direction for a young interior design studio targeting creative urban clients.
```

## Expected Behaviour

ORBIT should identify the module, explain the route and produce the next useful output.
