# ChatGPT Runtime

This folder makes ORBIT usable directly inside ChatGPT.

## Purpose

The repository is the knowledge base. This folder provides the practical prompts and instructions needed to run ORBIT in a normal ChatGPT conversation or in a custom GPT.

## Use Cases

- Use ORBIT inside a normal ChatGPT thread
- Build an ORBIT custom GPT
- Start a new client or personal project
- Route requests to the right ORBIT module
- Keep outputs structured and consistent
- Create a brand kit with `Use_Case_Brand_Kit.md`

## Recommended Flow

1. Open `Start_Here.md`.
2. Paste the startup prompt into ChatGPT.
3. Add project context with `Orbit_Project_Intake.md`.
4. Use commands from `Runtime_Commands.md`.
5. Review outputs with `Runtime_Quality_Gate.md`.

## Brand Kit Workflow

Open `Use_Case_Brand_Kit.md` to create a complete brand kit with ORBIT.
