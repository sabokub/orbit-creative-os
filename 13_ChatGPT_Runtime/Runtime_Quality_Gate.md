# Runtime Quality Gate

Use this before accepting an ORBIT output.

## Quick Check

The output must be:

- specific
- structured
- useful
- consistent with the project
- based on the right module
- clear about assumptions
- actionable

## Revision Triggers

Revise the output if:

- it is generic
- it ignores the audience
- it starts with visuals before strategy
- it copies references without translation
- it gives decor ideas without spatial logic
- it creates posts without content pillars

## Approval Labels

Approved: ready to use.

Needs revision: useful but incomplete.

Blocked: missing context or wrong direction.

## Final Question

Can this output be used immediately by the user?

If not, revise it.
