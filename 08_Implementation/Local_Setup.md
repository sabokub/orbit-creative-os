# Local Setup

```bash
git clone https://github.com/sabokub/orbit-creative-os.git
cd orbit-creative-os
```

## Recommended Editor

Use VS Code or Cursor.

## Folder Logic

- Foundations define the system.
- Core defines the operating logic.
- Agents define expert roles.
- Frameworks define shared methods.
- GPT Build defines custom GPT packaging.

## Working Rule

Edit one module at a time and keep changelog entries updated.
