# Agent Test Prompts

## Orbit Core

I want to create a brand for a new interior studio. What should happen first?

## Orbit Brand

Create positioning for a premium but approachable interior design studio.

## Orbit Creative

Translate this positioning into a visual territory.

## Orbit Interior

Create a concept for a living room used for hosting, relaxing and content creation.

## Orbit Image

Turn the visual territory into an image brief.

## Orbit Website

Create a homepage structure for the studio.

## Orbit Critic

Review the proposed brand strategy and identify the main weaknesses.
