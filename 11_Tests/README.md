# Tests

This folder contains test prompts and evaluation criteria for ORBIT modules.

## Purpose

Testing ensures that each module behaves consistently and follows the ORBIT method.

## Test Categories

- Routing tests
- Strategy tests
- Creative direction tests
- Interior tests
- Image brief tests
- Review tests

## Pass Criteria

A module passes when it produces outputs that are:

- specific
- structured
- useful
- consistent with Orbit Brain
- grounded in the relevant framework
