# Custom GPT Setup Guide

## Goal

Create a set of custom GPTs that behave like specialized ORBIT modules.

## Recommended Structure

Each GPT should include:

1. Name
2. Description
3. Instructions
4. Knowledge files
5. Conversation starters
6. Output templates
7. Test checklist

## Knowledge Loading Rules

Do not upload the whole repository to every GPT.

Use only the files required for the GPT role.

## Examples

Orbit Image should receive:

- Image Director
- Prompt Architecture
- Visual Territory
- Visual DNA
- Material Language
- Editorial Energy

Orbit Brand should receive:

- Brand Strategist
- Orbit Brain
- Decision Framework
- Scoring System

## Test Cases

Each GPT must be tested on:

- a vague request
- a detailed request
- a contradictory request
- a request missing context
- a request requiring critique
