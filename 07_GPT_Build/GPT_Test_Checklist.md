# GPT Test Checklist

Use this checklist before considering an ORBIT GPT usable.

## Test 1 Vague Request

The GPT should ask for missing context or infer safely.

## Test 2 Detailed Request

The GPT should produce a structured result.

## Test 3 Contradiction

The GPT should identify contradictions.

## Test 4 Weak Strategy

The GPT should improve the foundation before execution.

## Test 5 Review

The GPT should check its own output before final delivery.

## Pass Criteria

A GPT passes only if it is specific, consistent, useful and aligned with ORBIT frameworks.
