# GPT Build Kit

This folder prepares ORBIT for implementation as custom GPTs.

## Purpose

The goal is to translate the ORBIT repository into reusable custom GPT configurations.

Each GPT should have:

- a clear role
- strict operating boundaries
- access to the right knowledge files
- standard input and output formats
- review rules
- testing criteria

## Planned GPTs

- Orbit Core
- Orbit Brand
- Orbit Creative
- Orbit Interior
- Orbit Image
- Orbit Website
- Orbit Content
- Orbit Critic

## Build Order

1. Orbit Core
2. Orbit Brand
3. Orbit Creative
4. Orbit Interior
5. Orbit Image
6. Orbit Critic

Orbit Core acts as the primary interface. The other GPTs are specialist modules.
