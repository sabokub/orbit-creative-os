# ORBIT Roadmap

## v1.0 Foundation

Goal: establish the repository structure and core concepts.

Deliverables:

- Repository architecture
- Orbit Brain specification
- Core principles
- Initial agent library
- Initial framework library

Status: in progress

## v1.1 Agent Expansion

Goal: turn each agent into a usable expert module.

Priority agents:

1. Brand Strategist
2. Creative Director
3. Interior Director
4. Image Director
5. Critic

Deliverables:

- Role definition
- Inputs and outputs
- Decision rules
- Checklists
- Failure patterns
- Templates

## v1.2 Framework Expansion

Goal: make shared frameworks detailed enough to support multiple agents.

Priority frameworks:

- Visual Territory
- Visual DNA
- Spatial Narrative
- Material Language
- Prompt Architecture
- Conversion Ladder

## v1.3 GPT Build Kit

Goal: prepare ORBIT for implementation as custom GPTs.

Deliverables:

- System instruction templates
- Knowledge file structure
- Agent packaging rules
- GPT setup guide
- Testing checklist

## v2.0 Interior Intelligence

Goal: create the first specialized professional vertical.

Focus:

- Interior design
- Decoration
- Spatial storytelling
- Materials
- Lighting
- AI image generation
- Client deliverables
