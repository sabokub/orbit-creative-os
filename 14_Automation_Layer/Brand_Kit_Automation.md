# Brand Kit Automation

## Goal

Generate a complete brand kit from one structured brief.

## Input Form Sections

### Project Basics

- project name
- activity
- offer
- audience
- market
- competitors

### Direction

- desired perception
- style keywords
- references
- colours to keep
- directions to avoid

### Deliverable

- website use
- social use
- client document use
- image prompts needed

## Automation Steps

### Step 1 — Strategy

Use Strategy Prompts to generate:

- diagnosis
- audience insight
- positioning
- promise
- messaging pillars

### Step 2 — Creative Direction

Use Creative Prompts to generate:

- visual territory
- visual DNA
- art direction rules
- red flags

### Step 3 — Brand Kit

Generate the complete kit:

- foundations
- verbal identity
- visual identity
- usage rules
- examples

### Step 4 — Image Prompts

Generate prompts for:

- moodboard
- hero image
- social post
- website section
- client document cover

### Step 5 — Review

Use Orbit Critic to score:

- clarity
- difference
- coherence
- recognition
- usability

### Step 6 — Export

Export as:

- Markdown
- Google Doc
- PDF-ready document

## Approval Rule

If the review score is under 40 out of 50, the workflow should generate corrections before final export.
