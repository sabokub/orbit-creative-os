# Interface Spec

## Goal

Define the minimum interface needed for a more complete ORBIT v1.

## Interface Type

A simple internal dashboard or no-code hub is enough for v1.

## Pages

### 1. Home

Shows available workflows:

- Brand Kit Generator
- Hero Image Review
- Interior Concept Generator
- Content Plan Generator

### 2. New Project

Form to create a project and collect brief information.

### 3. Project Workspace

Shows:

- brief
- generated outputs
- review report
- exports
- next actions

### 4. Workflow Runner

Guided screen with steps:

1. Input
2. Generate
3. Review
4. Export
5. Archive

### 5. Output Library

Stores previous outputs by project.

## Core Components

- Project cards
- Workflow selector
- Brief form
- Generate button
- Review button
- Export button
- Status label

## v1 UI Principle

The interface should guide decisions, not become a design tool.
