# Build Roadmap

## Phase 1 — Internal Automation Hub

Goal: reduce manual ChatGPT work.

Build:

- project intake form
- workflow selector
- brand kit workflow
- hero image review workflow
- export to document

## Phase 2 — Project Library

Goal: keep every project organized.

Build:

- project database
- output archive
- prompt archive
- review history

## Phase 3 — Lightweight Dashboard

Goal: replace the no-code interface if needed.

Build:

- project cards
- workflow runner
- output viewer
- export controls

## Phase 4 — Client-Ready Layer

Goal: prepare ORBIT for client work.

Build:

- intake links
- client-safe documents
- PDF exports
- reusable templates

## Recommended First Build

Start with Brand Kit Generator and Hero Image Review.

These two workflows create the most immediate value for 24March Studio.
