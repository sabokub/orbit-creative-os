# v1 Product Scope

## Goal

Make ORBIT usable without manually running every step in ChatGPT.

## v1 Product Shape

ORBIT v1 should behave like a guided automation hub.

The user starts from a form, chooses a workflow, uploads references if needed, then receives a structured deliverable.

## Main User Actions

- Create project
- Fill project brief
- Choose workflow
- Add references
- Generate output
- Review output
- Export result
- Archive project

## v1 Workflows

### 1. Brand Kit Generator

Creates brand strategy, creative direction, brand kit and review.

### 2. Hero Image Review

Scores a website hero image and suggests improvements.

### 3. Interior Concept Generator

Creates spatial diagnosis, concept, material direction and image prompt.

### 4. Content Plan Generator

Creates pillars, formats, calendar and reuse plan.

## v1 Deliverables

- Markdown output
- Google Doc output
- PDF-ready output
- Prompt pack
- Review report

## Not in v1

- User accounts
- Payment system
- Public client portal
- Full drag and drop editor
- Full design generation interface

## v1 Principle

Automate the repeated thinking, not every creative decision.
