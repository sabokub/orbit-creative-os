# ORBIT Automation Layer

## Purpose

This folder defines a more complete v1 for ORBIT: not only prompts and documentation, but a practical automation layer.

The goal is to reduce manual work inside ChatGPT while keeping the system simple enough to build quickly.

## v1 Automation Vision

A user fills one structured brief, then ORBIT runs a sequence of modules and generates usable outputs.

## Core Flow

1. Intake form
2. Project record
3. Strategy generation
4. Creative direction generation
5. Deliverable generation
6. Critic review
7. Export
8. Archive

## Recommended v1 Stack

- Form: Tally, Typeform, Google Forms or Notion Form
- Automation: Make or n8n
- AI: OpenAI API
- Storage: Notion, Google Drive or Airtable
- Export: Google Docs, PDF or Markdown
- Source of truth: GitHub repository

## v1 Workflows

- Brand Kit Generator
- Hero Image Review
- Interior Concept Generator
- Content Plan Generator
- Project Archive Workflow

## What v1 Should Not Do

- Full SaaS product
- Complex user accounts
- Public marketplace
- Advanced billing
- Overbuilt dashboard

## Success Criteria

ORBIT v1 automation is successful if it can turn one filled brief into a structured output with review and export, without manually copying every step in ChatGPT.
