# Workflow Blueprints

## 1. Brand Kit Generator

### Trigger

New project brief submitted.

### Steps

1. Save brief
2. Run strategy prompt
3. Run creative direction prompt
4. Generate brand kit structure
5. Generate image prompts
6. Run review prompt
7. Export final document
8. Archive project

### Output

- Brand kit document
- Prompt pack
- Review report

---

## 2. Hero Image Review

### Trigger

Hero image uploaded with project context.

### Steps

1. Read project context
2. Analyse hero image
3. Score image on 10 criteria
4. Identify risks
5. Generate improved prompt if needed
6. Export review

### Output

- Score out of 50
- Strengths
- Issues
- Corrections
- Approval status

---

## 3. Interior Concept Generator

### Trigger

Interior project brief submitted.

### Steps

1. Save space brief
2. Run spatial diagnosis
3. Generate concept
4. Generate material direction
5. Generate furniture logic
6. Generate image prompt
7. Run review
8. Export concept document

### Output

- Interior concept
- Material direction
- Image prompt
- Review report

---

## 4. Content Plan Generator

### Trigger

Brand strategy approved.

### Steps

1. Read strategy
2. Define content pillars
3. Generate content series
4. Create 30-day plan
5. Add reuse logic
6. Review plan
7. Export calendar

### Output

- Pillars
- Series
- 30-day plan
- Reuse plan
