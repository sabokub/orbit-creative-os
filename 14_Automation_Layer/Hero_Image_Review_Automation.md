# Hero Image Review Automation

## Goal

Review a website hero image automatically against brand and web criteria.

## Inputs

- project name
- brand context
- target audience
- image file or image description
- website goal
- text placement needs
- visual direction

## Review Criteria

Score each from 1 to 5:

1. brand alignment
2. visual impact
3. web readability
4. target audience fit
5. difference
6. editorial quality
7. desirability
8. visual territory coherence
9. hero section usability
10. risks and limits

## Steps

1. Read brand context
2. Analyse image
3. Score each criterion
4. Write strengths
5. Write issues
6. Recommend corrections
7. Decide if image is usable
8. Generate improved image prompt if needed

## Output

- total score out of 50
- detailed score table
- strengths
- problems
- corrections
- approval status
- improved prompt

## Approval Rule

45 to 50: approved

38 to 44: usable with corrections

Under 38: regenerate or heavily revise
