# Hero Image Review Prompt

Use this prompt to review a website hero image with ORBIT.

## Prompt

```text
/orbit-review
Analyse cette hero image pour le site de 24March Studio.

Note-la sur 50 selon :
1. alignement marque
2. impact visuel
3. lisibilite web
4. coherence avec la cible
5. differenciation
6. qualite editoriale
7. desirabilite
8. coherence avec le territoire visuel
9. utilisabilite comme hero section
10. risques et limites

Contexte marque :
24March Studio est un studio digital de decoration interieure et direction artistique d'interieur.
La vibe recherchee est lifestyle editorial a la maison, cool people live here, jeunes clients creatifs, interieurs personnels, photogeniques, colores, vivants, avec une energie mode, magazine et culture visuelle.

A eviter :
interieur beige trop sage, decoration classique, moodboard materiaux agence, nightlife, rendu IA generique, image trop vide ou trop lisse.

Donne :
- une note sur 50
- les points forts
- les problemes
- ce qu'il faut corriger
- si l'image peut etre utilisee en hero ou non
- une version amelioree du prompt image si besoin
```

## Review Logic

The image should feel like editorial lifestyle at home, not a generic decoration visual.

It should support the brand promise of 24March Studio: interiors with identity, personality and visual presence.
