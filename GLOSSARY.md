# Glossary

## Orbit Brain

The shared knowledge base that stores project context and decisions.

## Agent

A specialist module with a defined role and operating method.

## Framework

A reusable method for analysis, decision-making or output evaluation.

## Workflow

A repeatable sequence of steps for a project type.

## Quality Gate

A review point that prevents weak outputs from being delivered.

## Visual Territory

The repeatable visual language of a brand or project.

## Visual DNA

The stable visual traits that remain consistent across outputs.

## Spatial Narrative

The story and emotional journey created through space.

## Conversion Ladder

A model that helps move people from awareness to action.
